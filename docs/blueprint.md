# **App Name**: ShadowGrid

## Core Features:

- Interactive Map Display (GM): Display a modifiable tactical map on the GM screen for managing encounters and visualizing environments.
- Player View Display: Display player-specific information on a separate screen (table TV) for an immersive experience.
- Icon Library: An icon library for maps, placeable items, and character tokens.
- Moveable Items and Tokens: Moveable items and tokens.
- GM Screen Management: Provide a GM screen that allows the GM to control map settings, NPCs, environment and manage secret rolls and campaign information.

## Style Guidelines:

- Primary color: Dark purple (#4A148C) for a mysterious and futuristic feel.
- Background color: Very dark gray (#212121) for a modern and immersive look.
- Accent color: Electric blue (#03A9F4) for highlights and interactive elements.
- Body and headline font: 'Space Grotesk', a sans-serif with a computerized, techy feel.
- Use glowing, high-tech icons to represent different actions and elements.
- Design a split-screen layout for the GM and player views, optimizing the limited screenspace of each output device.
